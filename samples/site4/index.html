PocketGrid

Tiny and powerful CSS grid system
PocketGrid is a tiny and powerful grid system.
It offers many awesome features:
 

Features

Flexible

  • Unlimited number of columns
  • Unlimited nested grids
  • Automatic rows
  • Customisable gutters

Responsive

  • Unlimited number of breakpoints
  • Define your own breakpoints
  • Compatible with Media Queries and mobile device detection

Mobile-first

  • Optimized for mobile devices
  • Blocks are stacked by default
  • Instant load time

Content-first

  • Swap columns
  • Can define custom breakpoints for each content in your page

Semantic

  • Just define blocks and block groups in your HTML
  • Define columns in your CSS

Use it everywhere!

  • Very light: < 1 kb!
  • Compatible with Twitter Bootstrap
  • Most browsers supported (IE6+)
  • No need for jQuery or LESS/SASS
