admin on-line Help

How to manage your web site, e-mail accounts and other FREE Internet services offered by enmail.com

Follow the simple steps given below ……... to manage your Company’s Internet Presence by yourself.

Kind note that you can manage all the Internet Services offered by enmail.com only through your ‘admin’ access.


Steps for managing the Internet Services offered :
1. Log on to the Internet.

2. Type http://www.enmail.com/admin.html as the address in your browser

3. Type your web-site address and the password (given by enmail.com)

4. Manage your web site; create e-mail accounts, view current user details and change ‘admin’ password –as given below.


Manage web-site:
1. Click 'Manage Website' link.

2. You will be taken to 'admin' page of your web site. This page will display the files (i.e. web pages) already uploaded in your site-with the filename, date of upload and size of file.You will have the facility to rename the files uploaded or to delete the files. DO NOT rename the "index.html" file, which is the first page or home page of your web site.

3. You can add or replace the existing web site with multiple web pages according to your requirement-by the following steps.

4. Prepare the web pages you would like to add or replace, and store it in your PC.

5. Click "Browse" button appearing next to the box of 'File Name' and select the web page to be uploaded to your web site.

6. Click "Upload" button.

7. The web page you selected has now been uploaded in your web-site.

Similarly, you can upload multiple web pages to your web-site. Please ensure to create links between web pages to move from one web page to another.


Create e-mail accounts :
1. Click 'Create New User' link.

2. You will be taken to 'Create New User' page of your web site. This page will display a form for filling up New User details.

3. Fill the required details and click 'Add User'

4. An email account: <LoginName>@<yourclubname>. enmail.com will be created.

Similarly, you can create multiple e-mail accounts under your web-site. Please ensure to remember the password entered while creating e-mail accounts. Inform the user, the e-mail account name and password for him to use his e-mail account for communication.The user has option to change the password on accessing his account in the Internet, for privacy.


Show Users:
1. Click ‘Show Users’ link.

You will be taken to User Accounts of your web-site

2. The details of User Names created, their respective last login and quota used (i.e. space used in our Internet server) will be displayed.

3. You can ‘Edit’ or ‘Delete’ the Users listed by clicking on ‘Edit’ and ‘Delete’ link after checking the box given before the User Name.

4. You can 'Edit' or 'Delete' the Users listed by clicking on 'Edit' and 'Delete' link after checking the box given before the User Name. Choosing 'Edit' , you can modify the User name and other details including user password


Change ‘admin’ Password
1. Click ‘Change Password’ link.

2. You will be taken to ‘Change Password’ page of your web-site.

3. The page will display a form for filling up your existing password (shown as ‘Old Password’), new password and confirmation of new password (enter new password again).

4. Fill up the relevant details in the form and enter ‘Update’ button.

5. Now your ‘admin’ password would be changed with the new password entered by you.

Your access to the ‘admin’ account will now be with your new password only
